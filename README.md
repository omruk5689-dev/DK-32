# About
**DK-32 (Dark Knight 32)** is an open source ESP32 development board, based on the movie **The Dark Knight**, featuring its hero Batman and his Batarang.

The purpose of this project was to create an ESP32 board that would not be similar to any other board. Instead of using the typical rectangular form factor, the PCB was made in the shape of the Batarang silhouette.

The board uses **ESP32-WROOM** module inside and provides all the features typical to the regular ESP32 board – Wi-Fi, Bluetooth, various GPIO pins, PWM, ADC, SPI, I²C and UART support.

For all kinds of users, including makers and educators, DK-32 can become useful.

# Features

- 🦇 Custom Batman Batarang inspired PCB
- Inspired by **The Dark Knight**
- ESP32-WROOM module
- Built-in Wi-Fi and Bluetooth
- On-board USB-to-Serial converter
- Micro-USB connector for programming and power
- BOOT and RESET buttons
- On-board 3.3V voltage regulator
- Easily accessible GPIO headers
- Compact and clean component layout
- Fully open-source hardware

# PCB Design

The whole PCB was created from scratch using **EasyEDA Pro**.

Creating an unconventional PCB in the shape of the Batarang was an interesting task. The main task here was to squeeze all the required components into the Batarang shape without complicating the circuit board.

I tried my best to organize components and routes on the PCB to maintain neatness and still keep the idea of Batman in mind.

# Programing

Programming the DK-32 module is identical to programming a standard ESP32 module.

It can be done using:

- Arduino IDE
- PlatformIO
- ESP-IDF

With an integrated USB-to-Serial converter, the firmware upload process is made simple via the Micro-USB connector.
